Sistema Termo de Compromisso
SENAC, PE 2022.

Link: Sistema Termo de Compromisso

Quando um colaborador entra no SENAC, ele é registrado no banco de dados do RH e um responsável do RH imprime um documento para solicitar a criação de conta de rede deste colaborador. O mesmo assina o termo para que a equipe de Suporte do Senac possa criar a conta deste colaborador. O problema é que dessa forma se gasta muita folha de papel e demora o processo, e por isso que o sistema nasceu para auxiliar nestas questões.
O sistema de Termo de compromisso começa com a solicitação de um colaborador feita pelo RH, onde a mesma preenche os campos restantes de um formulário e o sistema envia automaticamente um e-mail para o colaborador. Ele recebe em seu e-mail e é redirecionado para uma tela e, após digitar o seu CPF correto ele pode ler e aceitar os termos de compromisso. Logo após isso, o sistema manda um e-mail para a equipe de Suporte do Senac. Por fim, um responsável do suporte entra no sistema, cria a conta de rede do colaborador a partir das informações recebidas e finaliza o processo, assim mandando e-mail com instruções para o colaborador.


Dashboards do Sistema Task Manager (Sistema de Chamados)
SENAC, PE 2023.

Meu coordenador de sistema me convocou para trabalhar em um novo projeto de sistema full-Stack: a criação do dashboard do Task Manager. O sistema Task Manager é o sistema de chamados do Senac, aonde os colaboradores criam chamados para a resolução de suas problemáticas. Entretanto, o sistema ainda não possuía uma forma clara de visualizar os dados gerados por este sistema e nem de tirar insights do sistema de chamados.
Diante disso, eu comecei o desenvolvimento do Dashboard do Task Manager. A sua produção demorou em torno de 1 mês, e a sua criação trouxe muitos benefícios para os colaboradores e gestores do SENAC. O sistema de Dashboard permitiu a visualização estratégica e gráfica dos dados de cada setor do SENAC, de seus colaboradores em relação aos chamados e o tempo médio na resolução dos chamados, por exemplo.


Sistema Projeto L.O.U.C.O (Envio de projetos)
SENAC, PE 2023.

Link: Sistema Projeto L.O.U.C.O

Meu coordenador de sistema me trouxe uma nova demanda de sistema, solicitada por outra unidade do SENAC: a criação de um sistema para o projeto L.O.U.C.O. Este projeto mencionado é uma modalidade do SENAC do Marco Zero, aonde alunos e professores poderão ter seus projetos avaliados pela internet. Ou seja, eles criam grupos e enviam seus arquivos dos projetos criados, afim que um avaliador dê um feedback.
O sistema Projeto L.O.U.C.O permite o envio e a avaliação de milhares de projetos sem o uso de papel, além de registrar tudo no banco de dados do SENAC.


Sistema Controle Financeiro
SENAC, PE Março de 2023.

Link: Controle Monetário

Eu desenvolvi este sistema individualmente como proposta de me desafiar os meus conhecimentos e desenvolver o meu próprio sistema. Neste cenário, decidi criar um sistema para ajudar com registros de gastos e de dinheiro que recebo de entrada. O foco do sistema foi no BACK-END e em exercitar o código funcional. Dessa forma, O sistema Controle monetário tem o foco em ajudar o usuário a registrar sua poupança e suas quinzenas de acordo com o ano. Ele permite você registrar o dinheiro que entrou e o dinheiro que saiu no mes selecionado.
O sistema está hospedado no servidor e pode ser acessado em qualquer lugar. Ele também possui o seu certificado SSL para que seja reconhecido como seguro pelo Google Chrome.
